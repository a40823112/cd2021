建立倉儲

Step1:在mdecourse的cmstemplate倉儲下按Use this template

Step2:在Repository name中輸入cd2021

Step3:在Y槽的tmp目錄下clone倉儲，在編輯器中輸入

git clone --recurse-submodules https://github.com/學號/cd2021.git

Step4:利用編輯器在cd2021目錄下輸入

python wsgi.py
